# SmartAvaas - Backend

This is the **Spring Boot backend** for the SmartAvaas Resident Community Management platform. It provides secure APIs for user authentication, rent payment, maintenance requests, amenity reservations, and more.

---

## 📦 Tech Stack
- Java 17
- Spring Boot
- Spring Security + JWT
- Spring Data JPA + PostgreSQL
- Maven
- REST APIs

---

## 🚀 Getting Started

### Prerequisites
- Java 17
- Maven
- PostgreSQL

### Clone and Setup
```bash
git clone https://github.com/<your-org>/smartavaas-backend.git
cd smartavaas-backend
mvn clean install
mvn sprin-boot:run
